# GSAP Image Trail Experience

## Overview

This project is an experimental interactive image trail experience built with JavaScript and GSAP.  
It explores how motion, timing, and user input can be combined to create a fluid and immersive visual interaction.

The experience reacts to mouse movement, generating a trail of images that appear, animate, and disappear smoothly, creating a dynamic sense of flow.

---

## Features

- Mouse-based image trail interaction
- Smooth animations powered by GSAP
- Custom animated intro text with word transitions
- Fully customizable behavior via GUI controls:
  - Image spawn frequency
  - Appear animation speed
  - Disappear animation speed
  - Delay before disappearance
- Custom cursor interaction
- Preloaded images for smooth rendering

---

## Technologies

- JavaScript Vanilla
- GSAP
- HTML / CSS
